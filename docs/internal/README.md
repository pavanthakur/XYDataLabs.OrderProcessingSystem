This subtree contains active internal operating material that is useful to maintain the repository but is not part of the main human-facing guide, reference, or curriculum trees.

Governance rule:

- Active operating trackers and live internal backlogs can remain here.

- Human-facing end-user guidance should live in `docs/guides/`, `docs/reference/`, or `docs/learning/` instead.
