```

│   ├── 1_MASTER_CURRICULUM.md  # ⭐ Day-by-day learning plan (Days 1-114)
```

## Quick Start

Open `Azure-Curriculum/1_MASTER_CURRICULUM.md` — that is the **single source of truth** for daily learning.

## Archive Policy

Files in `Archive/` are historical planning documents that have been superseded by active docs:

| Archived Content | Superseded By |
|---|---|
| `TODO/Azure-Migration/` | `ARCHITECTURE-EVOLUTION.md` + `1_MASTER_CURRICULUM.md` |
| `TODO/Microservices-Architecture/` | `ARCHITECTURE-EVOLUTION.md` (Phases 9-14) |
| `TODO/Technical-Enhancements/` | `ARCHITECTURE-EVOLUTION.md` + Docker/infra guides |
| `LearningHelp/DockerHelp.md` | `Documentation/02-Azure-Learning-Guides/DOCKER_COMPREHENSIVE_GUIDE.md` |
| `LearningHelp/sharedsettingsHelp.md` | `Documentation/03-Configuration-Guides/` |
| `Azure-Curriculum/Archive/` | Deleted — content consolidated into `1_MASTER_CURRICULUM.md` and `IMPLEMENTATION_NOTES.md` |

Archives are kept for historical reference. Do not update them.
